Scalable Generative Game Engine: Breaking the Resolution Wall via Hardware-Algorithm Co-Design

Wei Zeng, Xuchen Li, Ruili Feng, Zhen Liu, Fengwei An, Jian Zhao  ·  arXiv 2026

Memory Wall Breakthrough

Asymmetric resource allocation & memory-centric operator fusion on multi-accelerator clusters.

Abstract

We present a hardware-algorithm co-design framework that breaks through the "Memory Wall" bottleneck in generative game engines. Our approach features an asymmetric resource allocation strategy for sequence parallelism, a memory-centric operator fusion scheme, and manifold-aware latent extrapolation for temporal redundancy elimination. Validated on a cluster of programmable AI accelerators, the system achieves real-time 720×480 neural game frame generation with 50× pixel throughput improvement, running racing and platformer game benchmarks at 26.4/48.3 FPS with only 2.7ms equivalent latency.
